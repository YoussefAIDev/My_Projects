# Investigating Netflix Movies

This project explores a dataset of Netflix titles to focus specifically on **movies** and analyze trends in their durations, release years, and genres.

Using Python, Pandas, and Matplotlib, the project cleans the data, filters movies only, and visualizes key insights.
